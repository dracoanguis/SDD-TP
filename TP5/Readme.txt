TP5 HACHAGE

Pour ce TP, un fichier de statistique sur le nombre de collision avec commentaire généraux sur le développement as été crée sous le nom "statistique.log".
Le logiciel est un peu lent sur le début, cela vient du fait qu'on innitialise 100 000 types Personne qui est un type assez gourmand.
Il est possible de changer les données (Personne) passer au logiciel en changeant juste le fichier d'entré dans le programme, dans ce cas il faut juste faire
attention a la taille de la table juste qu'elle ne soit pas trop petite et aussie a ce que la forme du fichier sois la meme que , mais les statistiques s'acctualiseront automatiquement.

Le mot "collision" est ici compris comme le nombre de fois ou il est néccessaire de décaler un Hache et ainsi de modifier son code.

L'interface ne sert uniquement a indiqué les étapes de réalisation du programme.
